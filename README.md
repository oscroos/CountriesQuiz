# Countries Quiz

Map quiz app for practicing countries and U.S. states by selecting the prompted place on the map.

## Map Coverage

Some very small countries and island territories are not included because they are too small to click reliably at the current map scale.

Not included: Vatican City, Monaco, San Marino, Hong Kong, Macao, Madeira, Azores, Nauru, Tuvalu, Maldives, Seychelles, Comoros, French Polynesia, Palau, Micronesia, Marshall Islands, Tonga, and Kiribati.
